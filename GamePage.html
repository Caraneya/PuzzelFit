Gameplay Screen
  • Home Screen — Game name, today's challenge card, Play CTA, Calendar, Stats, Settings.
  • Loading — Brief loading state (max 2s) while gameplay screen preloads in background.
  • Welcome Popup — First-time greeting modal. CTA: "Let's go" (→ Tutorial) or "Skip".
  • Tutorial Overlay — 5-step inline overlay on first session. Ends with Today's Goal popup.
  • Today's Goal Popup — Shown after tutorial (first visit) or on loading into gameplay (returning). CTA: "Ready to play".
  • Gameplay Screen — Game board, score bar, timer, hint tooltip, toolbar (info, calendar, stats, settings).
  • Pause Sheet — Timer paused. Resume / Exit options. Triggered by tapping the timer.
  • Win Sheet — Congratulations, countdown to next puzzle, Share / Home CTAs.
  • Lose Sheet — Time elapsed, Try Again / Home CTAs.

Technical Notes
  • Stack: Vanilla HTML5 + CSS + JavaScript. No frameworks, no build tools. Opens directly in a browser.
  • Structure: Each game lives in its own subfolder. Shared styles: tokens.css + components.css. Per-game styles: base.css + game.css.
  • Responsive: 320px–1920px fluid full-screen. Portrait lock recommended.
  • Target devices: iOS Safari, Android Chrome. Minimum viewport 375px wide.
  • Performance: 60fps on mid-range devices (2019+). All animations use CSS transforms only — no layout thrash.
  • Touch handling: All interactions via touch/pointer events. No hover-only states. Minimum 44×44px touch targets.
  • Accessibility: WCAG AA contrast on all text. Interactive elements accessible via aria-label.
  • Icons: All icons rendered via icons.js — no raw SVG in HTML.

Sound Design

Synthesized audio via Web Audio API — no external files. Two shared modules at the project root:

  • sound-utils.js — SFX engine. Games call SoundUtils.play(id). Game-specific sounds registered via SoundUtils.register({}).
  • music-utils.js — Background music engine. Ambient D-major-pentatonic pads. State-reactive: tempo/pitch respond to timer urgency, win, and pause.
  • sounds.html — Interactive palette for auditioning and tuning every sound. Open in browser to adjust sliders.
