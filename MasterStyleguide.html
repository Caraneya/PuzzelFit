Shared Utilities (game-utils.js)

All games load game-utils.js after icons.js. Every function lives on the GameUtils namespace.

MethodSignatureNotes
navigateTo navigateTo(screenId) Fade between .proto-screen elements. Uses --duration-slow (400ms).
openPopup / closePopup (id) Adds/removes .is-open on a .overlay element.
openSheet / closeSheet (id) Adds/removes .is-open on a .sheet-overlay element.
switchSheet (fromId, action, delay = 100) Closes one sheet then runs action() after delayms. Default 100ms lets the close animation start before the next thing opens. Use TIMING.NAV_DELAY (400ms) when navigating away.
initSheetDismiss (onDismiss) Wires scrim-click dismiss for all .sheet-overlay elements. Skips those with data-no-dismiss (win/lose sheets).
showToast (toastId, msg, duration = 2200) Shows a toast for durationms. Pass a custom duration when the toast follows a sheet close — use TIMING.TOAST_DELAY (300ms) as the setTimeout wrapper delay, not this parameter.
makeTimer (groupEl, iconEl, displayEl) Returns { start, pause, reset, isRunning }. Renders pause/play icon automatically.
initSheetScrollbar (wrapId, thumbId) Custom scrollbar for sheet list areas. Init all sheet scrollbars in a loop: ['instr','cal','stats','settings','feedback'].forEach(n => GameUtils.initSheetScrollbar(`game-${n}-wrap`, `game-${n}-thumb`))
setTheme (theme, segmentId, storageKey) Sets data-theme on <html>, syncs segment UI, persists to localStorage.
initCalendar (prefix) Full calendar with prev/next navigation. Derives all element IDs from prefix (e.g. 'db' → 'db-cal-title').
initFeedbackForm (prefix) Star rating + checkbox. IDs derived from prefix.
initHomeDate (prefix) Sets date label and fills last-week day name rows on the home screen.
buildStreakGrid (gridId) Renders the 7-day streak calendar row inside a .cal-day grid.

TIMING constants — Each game should define a TIMING object at the top of its JS file so every setTimeout duration is named, not a magic number: 

const TIMING = {
  MERGE_1:          30,   // delay before first merge check
  MERGE_2:          80,   // cascaded merge re-check
  MERGE_ANIM:      360,   // merge animation hold
  FLOAT_REMOVE:    750,   // floating score label cleanup
  SHEET_OPEN:      600,   // sheet delay  — matches --duration-slow×1.5
  SHEET_SWITCH:    100,   // close → open gap (switchSheet default)
  NAV_DELAY:       400,   // navigate after sheet — matches --duration-slow
  POPUP_SHOW:      500,   // popup after screen transition
  POPUP_QUICK:     200,   // fast popup (e.g. post-tutorial)
  TOAST_DELAY:     300,   // toast after sheet close starts
  LOADING_DELAY:  2000,   // loading screen minimum
  HINT_AUTO_CLOSE: 5000,  // hint banner auto-dismiss
};
